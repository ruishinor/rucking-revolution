# Rucking Revolution Brand Build

This package is the independent 0-to-100 brand build for Rucking Revolution.

It was built from repo evidence, existing product ideas, article themes, and the explicit instruction to optimize for civilians first, not military positioning.

## Outcome

- Two complete brand directions for consideration
- A recommended lead direction for launch
- Website, ebook, Medium, and Instagram systems designed as one brand
- KDP package with current-process notes validated against official KDP help pages on 2026-03-29
- HTML demos and SVG logo concepts in `public/brand-build/`
- Restart instructions and module checklists in case a later session needs to continue the work

## Recommended Default

Choose `Version A - Quiet Momentum` if the immediate goal is to sell beginner ebooks, build trust fast, and make rucking feel welcoming to ordinary adults.

Keep `Version B - Carry Forward` as the sharper alternate if the brand later wants a more social-first, movement-building edge.

## Working Assumptions

- Primary buyer is a civilian beginner who wants simple, sustainable fitness and does not identify with tactical cosplay.
- Secondary buyer is a community-minded adult who wants structure, accountability, and shared challenge.
- The first book should be a beginner bridge product, not an advanced doctrine manual.
- The website is a trust engine, the ebook is the first paid conversion, Medium is the discovery engine, and Instagram is the repetition engine.
