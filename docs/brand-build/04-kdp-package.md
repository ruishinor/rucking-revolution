## Cover Design Brief

### Objective

Create a cover that looks credible, calm, and beginner-safe at Amazon thumbnail size.

### What The Cover Must Communicate

- this is for ordinary adults
- this is practical, not macho
- this is structured, not random
- this is a real book, not a quick PDF

### Audience

- adults 30-55
- new or returning exercisers
- people who dislike gyms or running
- civilians first

### Visual Direction

Preferred direction: `Quiet Momentum`

- warm light background
- dark pine or graphite title treatment
- clay accent or rule line
- simple route or pack symbol
- no photo collage
- no tactical imagery

### Thumbnail Rule

The title must remain readable when displayed small on Amazon mobile search results.

### KDP Cover Tech Notes

Per current KDP help:

- accepted cover file formats: JPEG or TIFF
- ideal ebook cover size: 2560 x 1600 px
- minimum size: 1000 x 625 px
- ratio target: at least 1.6:1
- color mode: RGB
- keep file under 50 MB

Use the KDP cover calculator for print editions:

- `https://kdp.amazon.com/us_us/cover-calculator`

## KDP Book Description

Use this as the base description in KDP and adapt only if keyword data later suggests a tighter variant.

`Your First Ruck` is a beginner-friendly guide to one of the simplest and most effective ways to build strength, stamina, and consistency without turning your life upside down.

Rucking is just walking with weight. Done well, it can help you get fitter, stronger, and more resilient with less impact than running and less friction than the gym.

This book shows you how to start safely, progress steadily, and keep going for 30 days without overcomplicating the process.

Inside, you will learn:

- how much weight to carry as a beginner
- how to avoid the most common mistakes
- how to improve posture and reduce injury risk
- how to build a simple 30-day routine you can actually maintain
- how to track progress without obsessing over data
- what to do after your first month so the habit keeps growing

This is not a tactical fantasy book.

It is a practical guide for ordinary adults who want an effective, low-barrier way to get moving again.

If you have ever thought:

- "I need something simpler than the gym."
- "I want exercise I can actually stick with."
- "I do not want to feel behind before I even begin."

this book was written for you.

Start where you are. Carry what you can. Build the habit.

## Formatting Guide

### Recommended Workflow

1. Write and edit in Markdown or DOCX
2. Export to DOCX or EPUB
3. For ebook, prefer clean reflowable formatting
4. Preview in Kindle Previewer and KDP Online Previewer
5. Fix issues in the source file, then re-upload

### KDP Format Notes

KDP currently accepts multiple manuscript types for ebooks, including DOCX and EPUB. KDP also supports KPF files created in Kindle Create.

Recommended for this project:

- EPUB if the workflow is stable
- DOCX if the author wants the simplest editing pipeline
- Do not build the workflow around MOBI uploads

### Manuscript Rules

- use Heading 1 for chapter titles
- use Heading 2 for major section breaks
- avoid manual tabs and repeated spaces
- do not force page numbers in ebook files
- keep images sparse and purposeful
- keep callout styles simple enough to survive conversion
- include a linked table of contents

### Ebook Interior Rules

- chapter opener should be clean and high contrast
- body text should stay calm and readable
- no dense tables unless absolutely necessary
- worksheets should also work as plain lists if formatting collapses

### Front Matter

- title page
- copyright page
- optional short "how to use this book" note
- linked table of contents

### Back Matter

- short author note
- invitation to join the email list
- preview of next book in the series

### Quality Control

Before publishing:

- run spell check
- test the table of contents
- check grayscale rendering in preview
- inspect image quality warnings
- test chapter spacing on phone and e-reader views

## Submission Process

### 1. Book Details

- enter title, subtitle, series info if any
- set author and contributors
- paste description
- enter keywords
- choose categories
- set primary audience information if applicable

### 2. Book Content

- confirm publishing rights
- upload manuscript
- upload cover
- launch previewer
- review spell, image, and layout warnings

### 3. Pricing

- choose royalty option
- choose list price
- confirm territories
- review lending or KDP Select choice

### 4. Publish

- submit
- wait for review

Current official timing guidance indicates ebook review and publication are typically completed within 24 to 72 hours, and new ebook detail pages can take up to 72 hours to appear.
